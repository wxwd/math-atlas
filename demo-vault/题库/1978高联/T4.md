## 题目
已知方程 $2x^2 - 9x + 8 = 0$，求作一个二次方程，使它的一个根为原方程两根和的倒数，另一个根为原方程两根差的平方。

## 答案
（无）

## 解析
设方程 $2x^2 - 9x + 8 = 0$ 两根为 $x_1,x_2$，则 $x_1 + x_2 = \frac{9}{2}, x_1 x_2 = 4$。

令新方程的两根为 $x_1', x_2'$，于是 $x_1' = \frac{1}{x_1 + x_2} = \frac{2}{9}$，

$x_2' = \left(x_1 - x_2\right)^2 = \left(x_1 + x_2\right)^2 - 4x_1 x_2 = \frac{81}{4} - 16 = \frac{17}{4}.$

所以新方程为 $\left(9x - 2\right)\left(4x - 17\right) = 0 \Rightarrow 36x^2 - 161x + 34 = 0$。
