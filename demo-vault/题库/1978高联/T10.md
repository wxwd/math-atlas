## 题目
求方程组 $\left\{ \begin{array}{l} x + y + z = 0, \\ x^3 + y^3 + z^3 = -18 \end{array}\right.$ 的整数解。

## 答案
（无）

## 解析
$x^3 + y^3 + z^3 - 3xyz = \left(x + y\right)^3 + z^3 - 3xy\left(x + y\right) - 3xyz$

$$= \left(x + y + z\right)\left[\left(x + y\right)^2 + z^2 - \left(x + y\right)z\right] - 3xy\left(x + y + z\right)$$

$= \left(x + y + z\right)\left(x^2 + y^2 + z^2 - xy - yz - zx\right) = 0 \Rightarrow x^3 + y^3 + z^3 = 3xyz,$

于是 $\left\{\begin{array}{l} x + y + z = 0, \\ xyz = -6 \end{array} \Rightarrow \left(x, y, z\right) = \left(1,2,-3\right),\left(1,-3,2\right),\left(2,1,-3\right),\left(2,-3,1\right),\right.$，

$\left(-3,1,2\right),\left(-3,2,1\right)$ 共 6 组解。
