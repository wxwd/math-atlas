## 题目
已知 $\tan x = 2\sqrt{2}\left(180^\circ < x < 270^\circ\right)$，求 $\cos 2x,\cos \frac{x}{2}$ 的值。

## 答案
（无）

## 解析
由于 $180^\circ < x < 270^\circ$，则 $\tan x = 2\sqrt{2} \Rightarrow \cos x = -\frac{1}{3}$，

所以 $\cos 2x = 2\cos^2 x - 1 = -\frac{7}{9};\cos \frac{x}{2} = -\sqrt{\frac{1 + \cos x}{2}} = -\frac{\sqrt{3}}{3}$。
