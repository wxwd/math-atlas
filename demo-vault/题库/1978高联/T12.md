## 题目
(1) 分解因式：$x^{12} + x^9 + x^6 + x^3 + 1$；

(2) 证明：对于任意角度 $\theta$，都有 $5 + 8\cos \theta + 4\cos 2\theta + \cos 3\theta \geq 0$。

## 答案
（无）

## 解析
(1) $x^{12} + x^9 + x^6 + x^3 + 1 = x^6\left(x^6 + \frac{1}{x^6} + x^3 + \frac{1}{x^3} + 1\right) \; = x^6\left[\left(x^3 + \frac{1}{x^3}\right)^2 + x^3 + \frac{1}{x^3} - 1\right] \; = x^6\left(x^3 + \frac{1}{x^3} - \frac{\sqrt{5} - 1}{2}\right)\left(x^3 + \frac{1}{x^3} + \frac{\sqrt{5} + 1}{2}\right) \; = \left(x^6 - \frac{\sqrt{5} - 1}{2}x^3 + 1\right)\left(x^6 + \frac{\sqrt{5} + 1}{2}x^3 + 1\right).$

(2) $5 + 8\cos \theta + 4\cos 2\theta + \cos 3\theta = 5 + 8\cos \theta + 8\cos^2\theta - 4 + 4\cos^3\theta - 3\cos \theta \; = 4\cos^3\theta + 8\cos^2\theta + 5\cos \theta + 1 = \left(\cos \theta + 1\right)\left(4\cos^2\theta + 4\cos \theta + 1\right)$

$= \left(\cos \theta + 1\right)\left(2\cos \theta + 1\right)^2 \geq 0.$
