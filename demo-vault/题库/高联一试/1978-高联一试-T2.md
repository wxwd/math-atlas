## 题目

已知 $\tan x = 2\sqrt{2}$（$180^\circ < x < 270^\circ$），求 $\cos 2x$，$\cos \dfrac{x}{2}$ 的值。

## 答案

$\cos 2x = -\dfrac{7}{9}$，$\cos \dfrac{x}{2} = -\dfrac{\sqrt{3}}{3}$。

## 解析

由于 $180^\circ < x < 270^\circ$，则 $\tan x = 2\sqrt{2} \Rightarrow \cos x = -\dfrac{1}{3}$，

所以 $\cos 2x = 2\cos^2 x - 1 = -\dfrac{7}{9}$；$\cos \dfrac{x}{2} = -\sqrt{\dfrac{1 + \cos x}{2}} = -\dfrac{\sqrt{3}}{3}$。
