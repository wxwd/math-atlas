## 题目

证明：当 $n, k$ 都是给定的正整数，且 $n > 2, k > 2$ 时，$n(n - 1)^{k - 1}$ 可以写成 $n$ 个连续偶数的和。

## 答案

$n(n - 1)^{k - 1}$ 可以写成从 $(n - 1)^{k - 1} - (n - 1)$ 开始的 $n$ 个连续偶数的和。

## 解析

$2m + (2m + 2) + \cdots + [2m + 2(n - 1)] = 2mn + n(n - 1) = n(2m + n - 1)$

$\Rightarrow n(2m + n - 1) = n(n - 1)^{k - 1} \Rightarrow 2m = (n - 1)^{k - 1} - (n - 1)$，由于 $n > 2, k > 2$，则 $(n - 1)^{k - 1} - (n - 1)$ 为偶数 $\Rightarrow m = \dfrac{1}{2}[(n - 1)^{k - 1} - (n - 1)] \in \mathbf{N}^*$。

所以 $n(n - 1)^{k - 1}$ 可以写成从 $(n - 1)^{k - 1} - (n - 1)$ 开始的 $n$ 个连续偶数的和。
