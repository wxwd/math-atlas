## 题目

证明：顶点在单位圆上的锐角三角形的三个角的余弦的和小于该三角形的周长之半。

## 答案

$\cos A + \cos B + \cos C < \sin A + \sin B + \sin C$。

## 解析

$a = 2\sin A, b = 2\sin B, c = 2\sin C$，本题即证：

$\cos A + \cos B + \cos C < \sin A + \sin B + \sin C$。

由于 $\triangle ABC$ 为锐角三角形，则 $A + B > \dfrac{\pi}{2}$，$0 < A, B < \dfrac{\pi}{2} \Rightarrow A > \dfrac{\pi}{2} - B$

$\Rightarrow \sin A > \cos B$，同理可得 $\sin B > \cos C$，$\sin C > \cos A$。

三式相加即得 $\sin A + \sin B + \sin C > \cos A + \cos B + \cos C$。
