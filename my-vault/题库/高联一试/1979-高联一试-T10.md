## 题目
已知锐角三角形的三边 $a, b, c$ 满足不等式 $a > b > c$，问四个顶点都在三角形边上的三个正方形哪个面积最大？证明你的结论。

## 答案
当 ${PQ}$ 在 ${AB}$ 边上时，内接正方形的面积最大。

## 解析
![2_972_1527_419_321_0.jpg](images/2_972_1527_419_321_0.jpg)

如图，$\bigtriangleup {ASE} \sim  \bigtriangleup {SBP} \Rightarrow  \dfrac{SE}{BP} = \dfrac{AE}{SP}$

$\Rightarrow  {SE} = \dfrac{yt}{x}$，同理可得 ${ER} = \dfrac{zt}{x}$。

于是 $\dfrac{yt}{x} + \dfrac{zt}{x} = x \Rightarrow  \left( {y + z}\right) t = {x}^{2}$

$\Rightarrow  \left( {a - x}\right) \left( {{h}_{a} - x}\right)  = {x}^{2} \Rightarrow  x = \dfrac{a{h}_{a}}{a + {h}_{a}}$

$= \dfrac{2S}{a + \dfrac{2S}{a}}$，即当 ${PQ}$ 在 ${BC}$ 边上时，内接正方形的边长 ${a}_{1} = \dfrac{2S}{a + \dfrac{2S}{a}}$。同理可得，当 ${PQ}$ 在 ${AC}$ 边上时，内接正方形的边长 ${a}_{2} = \dfrac{2S}{b + \dfrac{2S}{b}}$，

当 ${PQ}$ 在 ${AB}$ 边上时，内接正方形的边长 ${a}_{3} = \dfrac{2S}{c + \dfrac{2S}{c}}$。如图，考虑函数 $f\left( x\right)  = x + \dfrac{2S}{x}\left( {x > 0}\right)$，若 $a > b > c > \sqrt{2S}$，则 $f\left( a\right)  > f\left( b\right)  > f\left( c\right)$；否则，由于 ${bc} > {2S} \Rightarrow  c > \dfrac{2S}{b}$，于是 $f\left( b\right) \; = f\left( \dfrac{2S}{b}\right)  > f\left( c\right)$。所以当 ${PQ}$ 在 ${AB}$ 边上时，内接正方形的面积最大。
