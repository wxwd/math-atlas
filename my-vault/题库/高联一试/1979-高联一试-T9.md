## 题目
已知一点 $P\left( {3,1}\right)$ 及两直线 ${l}_{1} : x + {2y} + 3 = 0,{l}_{2} : x + {2y} - 7 = 0$，试求通过 $P$ 点且与 ${l}_{1},{l}_{2}$ 相切的圆的方程。

## 答案
${\left( x - \dfrac{4}{5}\right) }^{2} + {\left( y - \dfrac{3}{5}\right) }^{2} = 5$ 或 ${\left( x - 4\right) }^{2} + {\left( y + 1\right) }^{2} = 5$

## 解析
所求圆的圆心 $O$ 在直线 $x + {2y} - 2 = 0$ 上，且半径 $R = \sqrt{5}$，设 $O\left( {2 - {2t}, t}\right)$，则 ${\left| OP\right| }^{2} = {\left( 2t + 1\right) }^{2} + {\left( t - 1\right) }^{2} = 5{t}^{2} + {2t} + 2 = 5 \Rightarrow  5{t}^{2} + {2t} - 3 = 0 \Rightarrow  {t}_{1} = \dfrac{3}{5},{t}_{2} =  - 1$。所以圆的方程为 ${\left( x - \dfrac{4}{5}\right) }^{2} + {\left( y - \dfrac{3}{5}\right) }^{2} = 5$ 或 ${\left( x - 4\right) }^{2} + {\left( y + 1\right) }^{2} = 5$。
