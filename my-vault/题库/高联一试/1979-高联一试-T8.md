## 题目
设 $\bigtriangleup {ABC}$ 三内角成等差数列，三条对应边 $a, b, c$ 的倒数成等差数列，试求 $A, B, C$。

## 备注

### 我的备注

### AI备注



## 答案
$A=B=C=\dfrac{\pi}{3}$

## 解析
$\bigtriangleup {ABC}$ 三内角 $A, B, C$ 成等差数列 $\Rightarrow  B = \dfrac{\pi }{3}$，

而 $\dfrac{1}{a} + \dfrac{1}{c} = \dfrac{2}{b} \Rightarrow  \dfrac{1}{\sin A} + \dfrac{1}{\sin C} = \dfrac{2}{\sin B} = \dfrac{4}{\sqrt{3}} \Rightarrow  \dfrac{\sin A + \sin C}{\sin A\sin C} = \dfrac{4}{\sqrt{3}}$

$\Rightarrow  \dfrac{2\sin \dfrac{A + C}{2}\cos \dfrac{A - C}{2}}{\dfrac{1}{2}\left\lbrack  {\cos \left( {A - C}\right)  - \cos \left( {A + C}\right) }\right\rbrack  } = \dfrac{4\sqrt{3}\cos \dfrac{A - C}{2}}{2\cos \left( {A - C}\right)  + 1} = \dfrac{4\sqrt{3}}{3}$

$\Rightarrow  4{\cos }^{2}\dfrac{A - C}{2} - 3\cos \dfrac{A - C}{2} - 1 = 0 \Rightarrow  \cos \dfrac{A - C}{2} = 1$ 或 $- \dfrac{1}{4}$。

当 $\cos \dfrac{A - C}{2} =  - \dfrac{1}{4}$ 时，$\left| \dfrac{A - C}{2}\right|  > \dfrac{\pi }{2} \Rightarrow  \left| {A - C}\right|  > \pi$，矛盾；

所以 $\cos \dfrac{A - C}{2} = 1 \Rightarrow  A = C \Rightarrow  A = B = C = \dfrac{\pi }{3}$。
